Upgrading from b2evolution version 0.8.0-Beta3

If you are upgrading from 0.8.0-Beta3 instead of the 0.8.0 final relase, you need to delete the 'skin' cookie.

The new cookie is now called 'cookieb2evostate'. If you do not delete the 'skin' cookie, the skin switcher may still not work.

Then proceed with upgrading from b2evolution 0.8.0.
