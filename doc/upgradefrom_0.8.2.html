Upgrading from b2evolution version 0.8.2

As always, it is very wise to do a backup of your current website / blog / database (depending on what you already have) before starting to install something new.

So far, upgrading from version 0.8.2 requires you to:

  • Redo your /conf using the new files.
  • You can keep your skins & templates as they are but if you want to take advantage of localization, you should try the new ones.
  • Replace the application files (all the other files: admin, b2evocore, htsrv, xmlsrv, etc...) with the new ones
