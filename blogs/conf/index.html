evoCore FrameWork

/conf folder

This folder holds the configuration files for this evoCore application as well as this particular instance of the application.

This is a configuration folder, it only needs to be accessed by PHP. It does not need to be accessed through the web.

See the comments in each file for details about configuration options.

There are several levels of settings, which can sometimes override each other:

  • The conf settings for the application
  • The conf settings for a particular instance of the application
  • The general settings in the backoffice
  • The user settings in the user profile

Note: in order to do local testing with different settings than your server settings, you can also override some of your conf settings with files named _config_TEST.php and _overrides_TEST.php .
