/skins_adm folder

This folder holds the skins for the admin interface (the back-office).

The adminUI_general class implements a default behaviour for the admin User Interface. This default behaviour can be overridden in different skin subfolders.

b2evolution Note: this folder only pertains to the admin skins. The skins for the public blog views are to be found the the /skins folder.

This folder needs to be accessed through the web.
